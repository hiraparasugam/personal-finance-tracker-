# 💰 AI Personal Finance Tracker

## Overview

An AI-powered finance dashboard built with Python, Streamlit, Pandas, Plotly, and Scikit-Learn.

The application analyzes bank statement CSV files, categorizes transactions automatically, visualizes spending patterns, generates financial insights, and predicts future expenses.

---

## Features

✅ Upload bank statement CSV

✅ Automatic transaction categorization

✅ Income, Expense, and Savings tracking

✅ Interactive expense distribution charts

✅ Category-wise spending analysis

✅ Top expense identification

✅ AI-powered financial insights

✅ Monthly spending trend analysis

✅ Expense forecasting using Machine Learning

✅ Downloadable expense reports

---
